简介

简单地说 Node.js 就是运行在服务端的 JavaScript。
Node.js 是一个基于 Chrome JavaScript 运行时建立的一个平台。
Node.js 是一个事件驱动 I/O 服务端 JavaScript 环境（单线程），基于 Google 的 V8 引擎，V8 引擎执行 Javascript 的速度非常快，性能非常好。
